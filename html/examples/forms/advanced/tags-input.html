Tags Input（标签输入） 官网

文本框

只需要在文本框上添加 data-role="tagsinput" 即可

多个值

可以通过 <select multiple> 来获得多个值，而不是一个文本的字符串， <option> 会自动被设置成一个标签，用英文的逗号隔开。

对象（Object）

绑定对象而非字符串到标签上，这样在某些场景下会比较灵活

标签分类

您可以使用静态的css类，或动态的函数来设置标签的分类
