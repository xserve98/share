添加行

本示例演示了通过点击按钮添加一条数据到表格中

row.add() 添加一行

rows.add()添加多行（注意多了一个s）

需要注意的是，你调用这个方法后需要重绘表格（调用 draw()）才能看到效果

第1列 第2列 第3列 第4列 第5列
第1列 第2列 第3列 第4列 第5列
  • JavaScript
  • HTML 
var t = $('#dataTableExample').DataTable($.concatCpt('dataTable'));
var counter = 1;

$('#DTddRow').on('click', function () {
    t.row.add([
        counter + '.1',
		counter + '.2',
		counter + '.3',
		counter + '.4',
		counter + '.5'
	]).draw(false);

	counter++;
});

// 自动添加第一行的数据
$('#DTddRow').click();
